# Resume Bullets

- Built MoneyTrack AI, a full-stack finance dashboard using Next.js 15, TypeScript, FastAPI, SQLite, Pandas, and NumPy to track income, expenses, cashflow, and financial risk.
- Implemented a rule-based AI financial advisor that flags negative cashflow, high debt burden, low savings rate, category concentration, and business margin risk without paid APIs.
- Designed a responsive SaaS-style dashboard with Recharts visualizations, dark mode, bilingual English/Thai labels, transaction CRUD, and a what-if simulator for spending and income scenarios.
- Added backend pytest coverage for financial calculations, advisor rules, health score logic, and what-if simulation, plus frontend lint, typecheck, and production build verification.
